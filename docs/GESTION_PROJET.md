# Documentation — Gestion de projet (ECF)

## Outil utilisé
**Trello** (ou Notion/Jira selon le lien fourni).

## Organisation
- **Backlog** : demandes non planifiées
- **À faire** : tâches à réaliser
- **En cours** : tâches en développement
- **En review/test** : vérification fonctionnelle
- **Terminé** : tâches validées

## Méthode
- Découpage par fonctionnalités (menus, commandes, avis, admin).
- Chaque carte contient : description, critères d’acceptation, sous‑tâches.
- Vérification à la fin de chaque fonctionnalité.

## Suivi et validation
- Tests manuels sur parcours utilisateur.
- Validation par rôle (USER, EMPLOYE, ADMIN).
